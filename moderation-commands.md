# Moderation Commands

&lt;&gt; = Required  
\[\] =Optional

### Ban

```text
Ban Command:
/ban <user> [reason]
```

### Kick

```text
Kick Command:
/kick <user> [reason]
```

### Warn

```text
Warn Command:
/warn <action> <user> [reason]
```

### Purge Message

```text
Purge Command:
/purge <number of messages>
```
